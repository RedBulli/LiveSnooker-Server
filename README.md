LiveSnooker-Server
==================
Installation
------------
```
npm install
```

Run development server
------------
Copy environments/sample.env to environments/development.env and modify values (defaults might also work for you). Then run:
```
./grunt.sh serve
```

Testing
-------
Run:
```
./grunt.sh test
```

Or to run tests when files change:
```
./grunt.sh tdd
```
